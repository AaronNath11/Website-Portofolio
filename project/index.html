Aplikasi Penjualan Barang Berbasis QR Code

Sebagai bagian dari eksplorasi saya dalam pengembangan aplikasi, saya berhasil merancang dan mengembangkan aplikasi penjualan barang menggunakan MIT App Inventor. Aplikasi ini memungkinkan pengguna untuk menyimpan barang dengan menggunakan QR code untuk setiap item. Dengan fitur ini, pengguna dapat dengan mudah menambahkan, mengupdate, dan melacak stok barang secara efisien. Proyek ini tidak hanya memperdalam keterampilan teknis saya dalam pemrograman aplikasi mobile tetapi juga memberikan wawasan tentang pengembangan teknologi QR code.
